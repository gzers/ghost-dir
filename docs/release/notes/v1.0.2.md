# 🔧 Ghost-Dir 1.0.2

> 发布日期：2026-02-26

Bug 修复版本，修复了"新增链接"对话框中模板搜索与分类筛选失效的问题。

## 🐛 Bug 修复

- **修复"新增链接"模板名称搜索失效**：`TemplateManager` 缺少 `search_templates` 方法，导致文本框输入关键词后列表无响应，现已补充实现，支持按模板名称和描述字段模糊匹配。
- **修复"新增链接"分类筛选失效**：`TemplateManager` 缺少 `get_templates_by_category_recursive` 方法，导致选择分类后无法过滤模板列表，现已补充实现，支持递归匹配所有子孙分类。
- **修复选择分类时崩溃**：`CategorySelector._on_category_selected` 直接访问 `CategoryNode.full_path_name` 属性，该属性在模型中不存在，导致 `AttributeError`，已改为防御性 `getattr` 访问。
