## 📝 本次更新概要

1.0.1 是 1.0.0 正式版后的首个修订版本，专注于 **Bug 修复**与**打包流程优化**，不包含新增功能或破坏性变更。

---

## 🐛 修复 (Fixed)

### 迁移循环 Bug
修复了链接迁移过程中，当源路径被正在运行的进程锁定时（如 `razer_elevation_service.exe`），系统触发 `TARGET_EXISTS` 错误后反复弹出迁移确认对话框、导致无限循环的问题。

### 链接冲突处理
修复了建立链接时，若目标软件路径已包含数据会导致操作失败的问题。现在会提供三种选择：
- **覆盖**：清除已有数据并创建链接
- **迁移**：将已有数据迁移至仓库路径后再创建链接
- **取消**：中止操作

### 启动画面主题适配
修复了启动画面（Splash Screen）背景色未跟随系统主题（明/暗模式）切换的问题，同时新增 Mica 半透明效果支持。

---

## ⚡ 优化 (Changed)

### PyInstaller 打包配置
修复了打包为可执行文件后，部分模块因缺少隐藏导入（`hiddenimports`）配置而出现 `ModuleNotFoundError` 的问题。完善了 `Ghost-Dir.spec` 文件的 `hiddenimports` 列表。

### 配置文件打包方式
将 `config` 目录（分类配置、模版配置、UI 设置）直接通过 `.spec` 文件的 `datas` 字段嵌入打包产物，移除了之前的运行时自动初始化逻辑。打包后的程序开箱即用，无需手动复制配置文件。

---

## 📦 安装与升级

### 从 1.0.0 升级

1. 下载 `Ghost-Dir-1.0.1-win64.zip`
2. 解压后替换旧版目录
3. 原有数据（连接、模版）自动保留

### 从源码运行

```bash
git pull origin main
python run.py
```
