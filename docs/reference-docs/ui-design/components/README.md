## 📂 组件库

### [qfluent-widgets/](./qfluent-widgets/) - QFluentWidgets组件库

基于 PyQt-Fluent-Widgets 的 Fluent Design 风格组件库:

> 说明：该目录属于第三方参考镜像，状态应视为 `external`，不作为项目实现规范。

- **[基础输入组件](./qfluent-widgets/basic-input/)** - 按钮、输入框、选择器等
- **[导航组件](./qfluent-widgets/navigation/)** - 导航栏、标签页、面包屑等
- **[对话框组件](./qfluent-widgets/dialogs/)** - 对话框、消息框、浮出层等
- **[状态信息组件](./qfluent-widgets/status-info/)** - 进度条、信息栏、提示等
- **[文本组件](./qfluent-widgets/text/)** - 标签、文本编辑器等
- **[视图组件](./qfluent-widgets/views/)** - 列表、表格、树形视图等
- **[窗口组件](./qfluent-widgets/windows/)** - Fluent窗口、分割窗口等
- **[日期时间组件](./qfluent-widgets/date-time/)** - 日期选择器、时间选择器等
- **[布局组件](./qfluent-widgets/layout/)** - 流式布局等
- **[菜单组件](./qfluent-widgets/menus/)** - 菜单、系统托盘菜单等
- **[滚动组件](./qfluent-widgets/scroll/)** - 滚动区域、平滑滚动等
- **[材料组件](./qfluent-widgets/material/)** - 亚克力效果等
- **[媒体组件](./qfluent-widgets/media/)** - 视频播放器等

查看 [QFluentWidgets 组件库主页](./qfluent-widgets/README.md) 了解详情。
更多外部资料入口见 [external/README.md](../../external/README.md)。

## 🚀 使用指南

### 查找组件

1. 浏览 [qfluent-widgets/](./qfluent-widgets/) 目录
2. 选择对应的组件分类
3. 查看 `examples/` 目录中的示例代码

### 运行示例

```bash
cd docs/reference-docs/ui-design/components/qfluent-widgets/basic-input/examples/

python button_demo.py
```
